• Блочные:
    1. <h1> — Задает заголовок первого уровня.
    2. <h2> — Задает заголовок второго уровня.
    3. <h3> — Задает заголовок третьего уровня.
    4. <h4> — Задает заголовок четвёртого уровня.
    5. <h5> — Задает заголовок пятого уровня.
    6. <h6> — Задает заголовок шестого уровня.
    7. <ul> — Устанавливает маркированный список.
    8. <table> — Создает таблицу.
    9. <p> — Определяет текстовый абзац.
    10. <ol> — Устанавливает нумерованный список.
    11. <hr> — Рисует горизонтальную линию.
    12. <form> — Устанавливает форму на веб-странице.
    13. <fieldset> — Предназначен для группирования элементов формы.
    14. <div> — Универсальный блочный элемент.
    15. <pre> — Определяет блок предварительно форматированного текста.
    16. <menu> — Отображает список меню.
    17. <isindex> — Используется для поискового индекса в текущем документе.
    18. <center> — Выравнивает содержимое контейнера по центру относительно родительского элемента.
    19. <blockquote> — Предназначен для выделения длинных цитат внутри документа.
    20. <address> — Предназначен для хранения информации об авторе
  • Строчные:
